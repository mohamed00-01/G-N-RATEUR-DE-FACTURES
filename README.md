# GENERATEUR-DE-FACTURES
Développement d’une application web permettant de créer et télécharger des factures au format PDF.
 Conception d’une interface utilisateur en HTML/CSS avec formulaires pour
la saisie des informations (client, produits, quantités, prix).
 Implémentation de la logique de calcul en JavaScript (total HT, TVA, total
TTC).
 Génération dynamique d’un aperçu de facture avec mise en forme professionnelle (DOM + CSS).
 Intégration d’une librairie front-end pour l’export en PDF (jsPDF).
 Optimisation de l’ergonomie et de la lisibilité afin de garantir une expérience
utilisateur fluide.
Technologies: HTML, CSS, JavaScript, jsPDF
